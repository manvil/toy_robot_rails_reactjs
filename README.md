# ToyRobot in Rails and ReactJS

This is a simple application of a toy robot using Rails and React JS.

### Applications used
* Ruby / Rails
* ReactJS
* Webpacker
* Rspec
* Jest/Enzyme


## Installing
 * Install ruby 2.5.0
 * Setup bundler
 * bundle install


## Getting Started

Run
 * bundle exec rails s
 * Login to your web brower at http:/localhost:3000

## Running the tests

 * bundle exec rspec
 * npm test

## Built With

* Sublime text
* Rails/Rubocop
* ReactJS
